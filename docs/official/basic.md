# Basic Example

This example demonstrates some of the basic features of React Router, including:

- Layouts and nested `<Route>`s
- Index `<Route>`s
- Catch-all `<Route>`s
- Using `<Outlet>` as a placeholder for child routes
- Using `<Link>`s for navigation
